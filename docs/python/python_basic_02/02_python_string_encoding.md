## 1 字符串使用

### 01 访问字符串中的值 下标/索引

> Python 不支持单字符类型，单字符在 Python 中也是作为一个字符串使用。

Python 访问字符串，使用`[]`来截取字符，下标从 0 开始，写法为`字符串[下表值]`，注意：从右往左数，下标从-1 开始，-1，-2，-3，...

```python
# 字符串索引参照表
正索引： 0   1   2   3   4   5
负索引：-6  -5  -4  -3  -2  -1
位  置： 1   2   3   4   5   6

# 字符串的截取的语法格式如下：
# 变量[头下标:尾下标] # 左闭右开区间

str = "STYLAN"
print(str[0])  # 输出第一个字符 S，str中其他字符的输出方式类似
print(str[:])  # 输出字符串全部字符
print(str[0:]) # 输出从下标 0 到字符串末尾的所有字符
print(str[:5])  # 输出从下标 0 到下标 5 的所有字符
print(str[2:5])  # 输出从下标 2 到下标 5 的所有字符
print(str[0:6:2])  # 输出从下标 0 到下标 6 的所有字符，步长为 2
print(str * 2)  # 输出字符串两次

```

### 02 转义字符

> 转义字符：反斜杠 `\`，将反斜杠后面的字符转义成特殊字符，例如`\n`表示换行，`\t`表示制表符

1. `\(在行尾时)` ：续行符
2. `\\`：反斜杠符号
3. `\'`：单引号
4. `\"`：双引号
5. `\a`：响铃
6. `\b`：退格(Backspace)
7. `\f`：换页符(FF)，将当前位置移到下页开头
8. `\n`：换行符
9. `\r`：回车，将 `\r` 后面的内容移到字符串开头，并逐一替换开头部分的字符，直至将 `\r` 后面的内容完全替换完成。
10. `\t`：横向制表符
11. `\v`：垂直制表符
12. `\ooo`：八进制数，例如：`\000`，`\141`，`\040`

```python
# 部分常用的转义
print("hello\nworld")
print("hello\tworld")
print("hello\"world\"")
print("hello\'world\'")
print("hello\\world")

# 使用 \r 实现百分比进度
import time

for i in range(101): # 添加进度条图形和百分比
    bar = '[' + '=' * (i // 2) + ' ' * (50 - i // 2) + ']'
    print(f"\r{bar} {i:3}%", end='', flush=True)
    time.sleep(0.05)
print()

```

### 03 字符串运算符

1. `+`： 字符串拼接
2. `*`：重复输出字符串
3. `[]`：通过索引获取字符串中字符
4. `[:]`：截取字符串，遵循左开右闭原则；截取字符串中的一部分，遵循左闭右开原则，`str[0:2]` 是不包含第 3 个字符的。
5. `in`：成员运算符 - 如果字符串中包含给定的字符返回 True
6. `not in`：成员运算符 - 如果字符串中不包含给定的字符返回 True
7. `r/R`：原始字符串，所有字符串直接使用，不考虑其他特殊用法 -> `print(R"\n\r\t")` -> `\n\r\t`
8. `%`：格式化字符串，将字符串中的占位符替换成指定值

```python

a = "Hello"
b = "Python"

print("a + b 输出结果：", a + b)
print("a * 2 输出结果：", a * 2)
print("a[1] 输出结果：", a[1])
print("a[1:4] 输出结果：", a[1:4])

if( "H" in a) :
    print("H 在变量 a 中")
else :
    print("H 不在变量 a 中")

if( "M" not in a) :
    print("M 不在变量 a 中")
else :
    print("M 在变量 a 中")

print (r'\n')
print (R'\n')

```

### 4 字符串内建函数

1. find() ：检测字符串中是否包含子字符串，如果包含子字符串返回开始的索引值，否则返回-1
2. count()：返回字符出现次数
3. index()：检测字符是否包含在字符串中，如果包含返回索引值，否则报错
4. replace()：替换
5. split()：以指定字符分割字符串，返回一个列表
6. startswith() ：是否以某个字符串开头，返回 True 或 False
7. endswith()：是否以某个字符串结尾，返回 True 或 False
8. capitalize()：首字母大写
9. lower()：小写字母转为大写
10. upper()：小写字母转为大写
11. strip()：去除字符串两端的空格
12. join()：将序列中的元素以指定的字符连接生成一个新的字符串
13. len()：返回字符串长度
14. removesuffix(): 删除字符串后缀

## 2 字符串编码

### 常见字符串编码类型

- ASCII：英文以及西欧语言，占 8bit / 1bytes
- GB2312：国家简体中文字符集，兼容 ASCII，占 2bytes
- Unicode：国际标准组织统一标准字符集，占 2bytes
- GBK：GB2312 的扩展字符集，支持繁体字兼容 GB2312，占 2bytes
- UTF-8：不定长编码，占 1~3bytes

### 字符串编码转换

1. encode() 编码
   将其他编码的字符串转换成 Unicode 编码
2. decode() 解码
   将 Unicode 编码转换成其他编码的字符串
