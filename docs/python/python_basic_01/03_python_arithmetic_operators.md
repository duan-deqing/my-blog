# 03 算术运算符

## Python 算术运算符：基本数学计算

包含：加(`+`)、减(`-`)、乘(`*`)、除(`/`)、整除(`//`)、取余(`%`)、幂运算(`**`)

1. 加(`+`)、减(`-`)、乘(`*`)：基本数学运算
2. 除(`/`) ：注意点除法得到的结果为浮点数。
3. 整除（`//`）: 前数除后数，结果为整数（向下取整）。
4. 取余 (%)：返回除法的余数（公式：`被除数 = 除数 * 商 + 余数`）
5. 幂运算 (`**`)：前数为底数，后数为指数（`a**b` 即 a 的 b 次方）,开平方（等价于 1/2 次方）

核心要点总结：

- 除法注意：`/` 始终返回浮点数，`//` 返回整数（向下取整，负数场景需特别注意）。
- 取余特性：余数的符号与除数一致（如 `10 % -3` 结果为 -2，因除数是负数）。
- 幂运算：`**` 不仅能算次方，还能通过 0.5 次方实现开平方（如 `25** 0.5` 得 5.0）。
- 复合运算：`+=、*=` 等可简化赋值语句，常用于循环中更新变量。
