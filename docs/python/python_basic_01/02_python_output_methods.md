# 02 Python 字符串格式化输出详解

三种主要方式：`% 格式化`、`str.format()`、`f-string`

## 1 % 格式化（旧版常用, 不推荐使用）

> % 格式化通过 `% 符号` 将变量嵌入字符串，需先定义字符串模板，再用 % 连接变量。不同类型的变量需搭配不同的 `格式符`（如 %s 对应字符串，%d 对应整数），类似 C 语言的 printf。

1. 核心语法

```python
# 单个变量
print("模板字符串%s" % 变量)

# 多个变量（需用括号包裹，顺序与模板内格式符对应）
print("模板1%s，模板2%d" % (变量1, 变量2))

```

2. 常用格式符

- `%s` 输出字符串（万能符，可接任何类型）。如，`"姓名：% s" % "张三"`
- `%d` 输出整数。如，`"年龄：% d" % 25`
- `%f` 输出浮点数（默认保留 6 位小数）。如，`"身高：% f" % 1.75`
- `%.2f` 浮点数保留 2 位小数。如，`"体重：%.2f" % 62.30`

3. 实战示例

```python
# 1. 单个变量
name = "李四"
print("欢迎%s使用系统" % name)
# 输出：欢迎李四使用系统

# 2. 多个变量
age = 30
score = 89.5
print("%s的年龄是%d岁, 考试分数是%.1f分" % (name, age, score))
# 输出：李四的年龄是30岁, 考试分数是89.5分

```

## 2 str.format()方法(灵活通用)

> format 通过字符串的 format()方法传递变量，用{}作为占位符，支持按位置、按名称传递变量，灵活性强（兼容 Python 2 和 3）。

1. 核心语法

```python
# 1. 按位置传递（{}内可写索引，默认按顺序）
print("模板 1{}，模板 2{}".format(变量 1, 变量 2)) # 按顺序
print("模板 2{1}，模板 1{0}".format(变量 1, 变量 2)) # 按索引

# 2. 按名称传递（{}内写参数名，清晰不易错）
print("姓名：{name}，年龄：{age}".format(name=变量 1, age=变量 2))
```

2. 常用格式控制

与 f-string 类似，在{}内加:控制格式，例如：

- 浮点数保留 2 位：{变量:.2f}
- 整数补零：{变量:03d}

3. 实战示例

```python
# 1. 按位置传递
subject = "数学"
grade = 95.5
print("{}成绩：{:.1f}分".format(subject, grade))
# 输出：数学成绩：95.5 分
print("第 2 个变量：{1}，第 1 个变量：{0}".format(subject, grade))
# 输出：第 2 个变量：95.5，第 1 个变量：数学

# 2. 按名称传递（适合变量多的场景）
print("课程：{subj}，分数：{scr:.2f}，排名：{rank}".format(subj=subject, scr=grade, rank=3))
# 输出：课程：数学，分数：95.50，排名：3

# 3. 结合列表/字典使用
student = ["赵六", 22] # 列表
print("姓名：{0[0]}，年龄：{0[1]}".format(student))
# 输出：姓名：赵六，年龄：22（0 代表第一个参数 student）

```

## 3 f-string 格式化（推荐首选）

> f-string 是 Python 3.6 引入的 “格式化字符串字面值”，在字符串前加 f 或 F，直接用{变量名}将变量嵌入字符串，无需格式符，简洁且执行效率高。

1. 核心语法

```python
# 基础用法：{变量名}直接嵌入
print(f"固定文本{变量名}")

# 进阶用法：{变量名:格式} 控制输出格式（如保留小数、补零）
print(f"固定文本{变量名:.2f}")
```

2. 常用格式控制

- 浮点数保留 n 位小数：{变量:.nf}（n 为总位数）
- 整数补零（共 n 位）：{变量:0nd}（n 为总位数）
- 左 / 右对齐（共 n 位）：
  `{变量:<ns}`（左对齐）、`{变量:>ns}`（右对齐）、
  `{变量: ^ns}`(居中对齐)、`{变量:*^ns}`(居中对齐，用`*`填充)

3. 实战示例

```python
# 1. 基础用法
name = "王五"
age = 28
print(f"用户{name}今年{age}岁")
# 输出：用户王五今年28岁

# 2. 格式控制
height = 1.823
id_num = 5  # 希望输出3位（补零）
print(f"身高：{height:.2f}米，用户ID：{id_num:03d}")
# 输出：身高：1.82米，用户ID：005

# 3. 支持直接计算（无需额外定义变量）
a = 10
b = 20
print(f"{a} + {b} = {a + b}")  # 输出：10 + 20 = 30

text = 'Hello'
print("|{:10}|".format(text))  # 左右对齐 总长度为10
print("|{:<10}|".format(text)) # 左对齐 总长度为10
print("|{:>10}|".format(text)) # 右对齐 总长度为10
print("|{:^10}|".format(text)) # 居中对齐 总长度为10
print("|{:*^10}|".format(text)) # 居中对齐 用*填充空白
```
