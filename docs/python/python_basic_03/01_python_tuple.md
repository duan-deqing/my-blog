# 01 元组

## 1.0 元组核心特点

1. 不可变性

   - 创建后不能修改、添加、删除元素
   - 这使得元组更安全、性能更好

2. 创建方式

   ```python # 正确方式
    t1 = (1, 2, 3)
    t2 = 1, 2, 3
    t3 = (1,) # 单元素元组

    # 错误方式
    t4 = (1) # 这是整数，不是元组
   ```

3. 优势

   - 性能更好：比列表创建和访问更快
   - 数据安全：防止意外修改
   - 可哈希：可以作为字典键
   - 多返回值：函数返回多个值的理想选择

4. 适用场景
   - 存储不应改变的数据（如配置、常量）
   - 作为字典键
   - 函数返回多个值
   - 保证数据完整性

> 元组是 Python 中重要的不可变序列类型，在很多场景下比列表更合适！

## 1.1 元组的定义

元组（tuple）是 Python 中的一种数据类型，用于存储多个元素。
元组中的元素可以是不同类型的数据，并且元组中的元素是不可变的，即一旦定义了元组，就不能修改其中的元素。
定义空元组的方式是使用一对空的圆括号 `()`，无需在小括号中添加逗号。

元组的定义方式有两种：

1. 使用圆括号 `()` 包裹元素，元素之间用逗号 `,` 分隔。
2. 使用 `tuple()` 函数将其他数据类型转换为元组。

例如：

```python
# 定义一个包含整数的元组
tuple1 = (1, 2, 3, 4, 5)

# 定义一个包含不同类型元素的元组
tuple2 = (1, 'hello', 3.14, True)

# 定义单个元素的元组（必须加逗号）
single_tuple = (1,)  # 注意这里的逗号

# 将列表转换为元组
list1 = [1, 2, 3, 4, 5]
tuple3 = tuple(list1)

# 空元组
empty_tuple = ()
empty_tuple = tuple()
```

## 1.2 元组的访问

元组中的元素可以通过索引来访问，索引从 0 开始。例如：

```python
tuple1 = (1, 2, 3, 4, 5)

# 访问第一个元素
print(tuple1[0])  # 输出：1

# 访问最后一个元素
print(tuple1[-1])  # 输出：5
```

## 1.3 元组的常用操作

### 1.3.1 元组的拼接

元组可以通过 `+` 运算符进行拼接，例如：

```python
tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)

# 拼接两个元组
tuple3 = tuple1 + tuple2
```

#### 1.3.2 元组的重复

元组可以通过 `*` 运算符进行重复，例如：

```python
tuple1 = (1, 2, 3)

# 重复元组
tuple2 = tuple1 * 3
```

#### 1.3.3 元组的长度

可以使用 `len()` 函数获取元组的长度，例如：

```python
tuple1 = (1, 2, 3, 4, 5)

# 获取元组的长度
length = len(tuple1)
```

#### 1.3.4 元组的遍历

可以使用 `for` 循环遍历元组中的元素，例如：

```python
tuple1 = (1, 2, 3, 4, 5)

# 遍历元组中的元素
for element in tuple1:
    print(element)
```

#### 1.3.5 元组的切片

可以使用切片操作符 `:` 来获取元组的子集，例如：

```python
tuple1 = (1, 2, 3, 4, 5)

# 获取元组的子集
sub_tuple = tuple1[1:4]
```

#### 1.3.6 元组的删除

元组是不可变的，因此不能直接删除元组中的元素。但是可以通过 `del` 语句删除整个元组，例如：

```python
tuple1 = (1, 2, 3, 4, 5)

# 删除元组
del tuple1
```

## 1.4 元组与列表的区别

元组和列表都是 Python 中常用的数据类型，它们之间有一些区别：

1. 元组是不可变的，而列表是可变的。
2. 元组的访问速度比列表快，因为元组是不可变的。
3. 元组可以用于作为函数的参数和返回值，而列表则不能。
4. 元组可以用于作为字典的键，而列表则不能。

## 1.5 嵌套元组

元组可以嵌套，即元组中的元素可以是其他元组。例如：

```python
# 元组中可以包含其他元组
matrix = (
    (1, 2, 3),
    (4, 5, 6),
    (7, 8, 9)
)

print("矩阵:")
for row in matrix:
    print(row)

print(f"第二行第三列: {matrix[1][2]}")

# 混合嵌套
mixed = ("张三", 25, (80, 90, 85))
name, age, scores = mixed
print(f"学生: {name}, 年龄: {age}, 成绩: {scores}")
```

## 1.6 元组作为字典的键

元组可以作为字典的键，因为元组是不可变的。例如：

```python
# 列表不能作为字典键，但元组可以（因为不可变）
coordinates_dict = {
    (0, 0): "原点",
    (1, 2): "点A",
    (3, 4): "点B"
}

print("坐标字典:")
for coord, name in coordinates_dict.items():
    print(f"坐标{coord}: {name}")

# 查找坐标对应的点
print(f"坐标(1,2)对应: {coordinates_dict.get((1, 2))}")
```

## 1.7 总结

元组是 Python 中一种不可变的数据类型，用于存储多个元素。元组中的元素可以是不同类型的数据，并且元组中的元素是不可变的，即一旦定义了元组，就不能修改其中的元素。元组可以通过索引来访问，可以使用切片操作符 `:` 来获取元组的子集，可以使用 `len()` 函数获取元组的长度，可以使用 `for` 循环遍历元组中的元素。元组可以嵌套，即元组中的元素可以是其他元组。元组可以作为字典的键，因为元组是不可变的。

1. 不可变序列，创建后不能修改
2. 使用圆括号定义，单元素需要加逗号
3. 支持索引、切片、迭代等序列操作
4. 可以包含任意类型的元素
5. 可以作为字典的键
6. 性能比列表更好
7. 适合存储不应改变的数据
8. 常用于函数返回多个值
