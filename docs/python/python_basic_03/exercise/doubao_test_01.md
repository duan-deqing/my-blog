# 豆包练习题

## 练习题 1：列表基础操作

---

**题目**：创建一个列表 `numbers = [5, 2, 9, 1, 5, 6]`，完成以下操作：

1. 往列表末尾添加数字 `3`
2. 插入数字 `0` 到列表的第 1 个位置（索引 0）
3. 排序列表（升序）
4. 删除列表中最后一个元素
5. 打印最终的列表

## 练习题 2：元组的不可变性与应用

---

**题目**：定义一个元组 `person = ("张三", 25, "男", "程序员")`，完成以下操作：

1. 打印元组中第 2 个元素（年龄）
2. 尝试修改元组中的年龄为 `26`（观察会报什么错误）
3. 将元组转换为列表，修改年龄为 `26` 后再转回元组，打印结果

**提示**：元组元素不可直接修改，需通过 `list()` 转换为列表后修改，再用 `tuple()` 转回。

## 练习题 3：字典的增删改查

---

**题目**：创建一个存储书籍信息的字典 `book = {"title": "Python入门", "author": "李四", "price": 59}`，完成以下操作：

1. 打印书籍的作者
2. 将价格修改为 `49`
3. 新增一个键值对 `{"publisher": "编程出版社"}`
4. 删除键为 `author` 的键值对
5. 打印修改后的完整字典

**提示**：用 `dict[key]` 查 / 改值，新增直接赋值 `dict[new_key] = value`，删除用 `del dict[key]`。

## 练习题 4：集合的去重与交集

---

**题目**：有两个列表 `list1 = [1, 2, 3, 4, 5, 3, 2]` 和 `list2 = [4, 5, 6, 7, 8, 5, 6]`，完成以下操作：

1. 将两个列表转换为集合，去除重复元素
2. 求两个集合的交集（同时存在于两个集合的元素）
3. 求两个集合的并集（所有元素，不重复）
4. 打印结果

**提示**：集合用 `set()` 转换，交集用 `&` 或 `intersection()`，并集用 `|` 或 `union()`。

## 练习题 5：列表推导式应用

---

**题目**：用列表推导式生成一个包含 `1-20` 中所有偶数的列表（即 `[2,4,6,...,20]`），并打印该列表。

**提示**：列表推导式格式 `[表达式 for 变量 in 范围 if 条件]`，条件为 `i % 2 == 0`。

## 练习题 6：字典遍历与筛选

---

**题目**：有一个学生成绩字典 `scores = {"张三": 85, "李四": 92, "王五": 78, "赵六": 95, "钱七": 60}`，完成以下操作：

1. 遍历字典，打印所有学生的姓名和成绩（格式：`张三：85分`）
2. 筛选出成绩大于等于 `90` 分的学生，存储到新字典 `excellent` 中并打印

**提示**：用 `for name, score in scores.items()` 遍历，筛选时用 `if score >= 90` 条件。

## 练习题 7：元组与列表的嵌套

---

**题目**：有一个嵌套结构 `students = [("张三", 20, ["数学", "英语"]), ("李四", 21, ["物理", "化学"])]`，完成以下操作：

1. 打印第一个学生的姓名和年龄
2. 打印第二个学生的第二门课程
3. 给第一个学生的课程列表添加一门 "语文"
4. 打印修改后的 `students`

**提示**：嵌套结构通过索引访问（如 `students[0][0]` 取第一个学生姓名），列表元素可修改。

## 练习题 8：集合的差集与对称差集

---

**题目**：有两个集合 `set_a = {1, 2, 3, 4, 5}` 和 `set_b = {4, 5, 6, 7}`，完成以下操作：

1. 求 `set_a` 相对于 `set_b` 的差集（属于 `set_a` 但不属于 `set_b` 的元素）
2. 求两个集合的对称差集（只属于其中一个集合的元素）
3. 向 `set_a` 中添加元素 `6`，再求一次对称差集，观察变化

**提示**：差集用 `-` 或 `difference()`，对称差集用 `^` 或 `symmetric_difference()`。
