# 豆包 python 函数练习题

## 练习题 1：基础函数定义与调用

---

**题目**：定义一个函数 `greet(name)`，接收一个字符串参数 `name`，返回形如 “你好，XX！” 的问候语（如调用 `greet("张三")` 返回 “你好，张三！”）。调用该函数，传入你的名字并打印结果。

**提示**：函数用 `def` 定义，返回值用 `return`，字符串拼接可用 `f-string`。

## 练习题 2：带默认参数的函数

---

**题目**：定义一个函数 `calculate_area(length, width=5)`，计算长方形面积（面积 = 长 × 宽），其中 `width` 有默认值 5。分别执行以下调用并打印结果：

1. `calculate_area(10)`（仅传长度，用默认宽度）
2. `calculate_area(10, 8)`（同时传长度和宽度）

**提示**：默认参数在参数列表后定义（`width=5`），调用时可省略默认参数。

## 练习题 3：多返回值函数

---

**题目**：定义一个函数 `get_max_min(numbers)`，接收一个包含整数的列表，返回列表中的最大值和最小值（以元组形式 `(max_num, min_num)` 返回）。调用该函数，传入列表 `[3, 1, 9, 5, 0]` 并打印结果。

**提示**：用 `max(numbers)` 和 `min(numbers)` 获取最大 / 最小值，`return` 后用逗号分隔两个值即返回元组。

## 练习题 4：关键字参数与不定长参数

---

**题目**：定义一个函数 `print_info(name, age, *hobbies, **extra)`，功能如下：

- 必选参数 `name`（姓名）、`age`（年龄）
- 不定长位置参数 `*hobbies`（爱好，可传 0 个或多个）
- 不定长关键字参数 `** extra`（额外信息，如 `city="北京"`）

调用函数：`print_info("李四", 28, "读书", "跑步", city="上海", job="教师")`，并在函数内打印所有信息（格式自定，需包含所有参数）。

**提示**：`*hobbies` 接收多余位置参数（元组形式），`**extra` 接收多余关键字参数（字典形式），遍历打印即可。

## 练习题 5：函数嵌套与作用域

---

**题目**：定义一个外层函数 `outer(a)`，该函数内部定义一个内层函数 `inner(b)`，`inner` 的返回值为 `a + b`。外层函数返回内层函数 `inner`。调用 `outer(5)` 得到内层函数，再用该函数传入 `3` 并打印结果（预期结果：8）。

**提示**：函数可作为返回值，内层函数可访问外层函数的变量（闭包特性）。

## 练习题 6：递归函数（阶乘计算）

---

**题目**：用递归函数 `factorial(n)` 计算 `n` 的阶乘（`n! = n × (n-1) × ... × 1`，规定 `0! = 1`）。调用该函数计算 `5!` 并打印结果（预期结果：120）。

**提示**：递归终止条件：`n == 0` 时返回 1；递归关系：`n! = n × (n-1)!`。

## 练习题 7：匿名函数（lambda）与内置函数结合

---

**题目**：有一个学生列表 `students = [("张三", 85), ("李四", 92), ("王五", 78)]`，其中每个元素是包含 “姓名” 和 “成绩” 的元组。用 `sorted()` 函数结合 `lambda` 按 “成绩从高到低” 排序该列表，并打印排序结果。

**提示**：`sorted()` 的 `key` 参数用 `lambda x: x[1]` 指定按成绩排序，`reverse=True` 表示降序。

## 练习题 8：函数作为参数（高阶函数）

---

**题目**：定义一个高阶函数 `operate(a, b, func)`，接收两个数字 `a`、`b` 和一个函数 `func`，返回 `func(a, b)` 的结果。分别传入以下函数作为 `func` 并打印结果：

1. 加法函数（`lambda x, y: x + y`）
2. 乘法函数（`lambda x, y: x * y`）

**提示**：函数可作为参数传递，调用时直接传入函数名或匿名函数。
